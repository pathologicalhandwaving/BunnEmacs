## Installation

Installed on macOS Catalina using Emacs28 HEAD

``` bash
brew tap daviderestivo/emacs-head
brew install emacs-head@28 --with-cocoa

git clone https://github.com/pathologicalhandwaving/BunnEmacs.git ~/.emacs.d
```
